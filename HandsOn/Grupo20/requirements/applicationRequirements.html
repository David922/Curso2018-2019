Application Requirements



What kind of information can show this application

This application try to show information of local establishment.



The application User interface will show :

  • Name of the cafe/restaurant
  • Which type of local establishment is (Takeaway Food Services,Cafes and Restaurants,Other Specialised Food Retailing, Pubs, Taverns and Bars)
  • Number of seats(Indoor or/and outdoor) for each local establishment
  • Address for each local establishment
